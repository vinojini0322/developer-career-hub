## Overview

Developer Career Hub is a production-grade SaaS platform designed to help software developers manage their career journey in one place. It enables users to track learning progress, manage job applications, record interview experiences, organize learning resources, and monitor career goals through an intuitive dashboard.

The application is being built using a microservices architecture with Spring Boot, React, PostgreSQL, Docker, and AWS following enterprise software development practices.

- Study progress
- Job applications
- Interview journals
- Resources
- Career goals
